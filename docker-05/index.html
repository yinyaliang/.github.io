    mysql中文显示问号

    查找docker

    docker ps
CONTAINER ID   IMAGE                                           COMMAND                  CREATED        STATUS         PORTS     NAMES
e12e03ea1e89   registry.eeo-inc.com/omdev/osp-mysql-init:v11   "docker-entrypoint.s…"   6 weeks ago    Up 3 minutes             osp-dev-8.0.20

    复制mysql配置文件

    cd /tmp
docker cp e1:/tmp/mysql/my.cnf .

    增加mysql配置

    [client]
default-character-set=utf8
 
[mysqld]
character-set-server=utf8
collation-server=utf8_general_ci

    替换mysql配置文件

    docker cp my.cnf e1:/tmp/mysql/my.cnf

    重启mysql

    docker stop e1
docker start e1
